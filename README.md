# Hello-Svelte
Demonstração de funcionalidades úteis do SvelteKit.

## Baixar o código:

```bash
git clone https://github.com/lasarinii/Hello-Svelte
```

## Executar:

```bash
npm install
npm run dev
```

Em seguida abra o navegador em (http://localhost:5173/)[http://localhost:5173/] para visualizar a página.
